Here are some artifact ideas for each topic:

**Understand a new codebase**
- Interactive dependency graph — paste a list of modules/classes and visualize how they connect
- Codebase onboarding checklist app — track which areas you've explored, add notes per module
- "Codebase radar" — rate familiarity across layers (DB, API, frontend, auth) and see a spider chart update live

**Refactoring legacy codebase**
- Mikado graph builder — visually map your refactoring prerequisites as a tree, check off nodes as you complete them
- Code smell → pattern matcher — pick a smell from a list, get the recommended pattern and a before/after code diff side by side
- Technical debt tracker — log files, rate severity, track status with a kanban-style board

**German A1–B1 review**
- Flashcard app with spaced repetition — build decks for vocab, cases, or verb conjugations
- Case trainer — interactive sentence builder that highlights Nominativ/Akkusativ/Dativ/Genitiv as you construct sentences
- Fill-in-the-blank story reader — paste a graded reader paragraph, blanks appear for key words, you type them in

**Learning DDD**
- DDD concept map — interactive diagram of Entities, Value Objects, Aggregates, Repositories, Domain Events with click-to-expand explanations
- Bounded context designer — drag-and-drop tool to sketch contexts and define relationships (ACL, Shared Kernel, etc.)
- Ubiquitous language builder — a living glossary where you define terms per context and Claude helps refine them

**Learning design patterns**
- Pattern refactoring simulator — shows messy code, you pick a pattern, it shows the refactored version with explanation
- Code smell → pattern decision tree — an interactive flowchart that walks you through choosing the right pattern
- Pattern cheat sheet — filterable card grid by category (Creational, Structural, Behavioral) with collapsible before/after examples

**Cybersecurity**
- Attack scenario simulator — pick a vulnerability (XSS, SQLi, IDOR...), walk through attacker steps and defensive countermeasures interactively
- Vulnerability checklist per layer — interactive audit checklist for your stack (PHP, MySQL, API, Vue) that tracks what you've hardened
- Race condition / business logic puzzle — a small payment scenario where you find the flaw and Claude confirms your answer
